# Portfolio Vika Zadorina

Сайт-портфолио фронтенд-разработчика с современным дизайном и удобным интерфейсом.

---

## О проекте

Проект демонстрирует навыки фронтенд-разработки:

- HTML5 с семантической разметкой  
- CSS3 Grid и Flexbox для адаптивной верстки  
- Стилизация карточек проектов и навыков  
- Использование Google Fonts и индивидуальной цветовой палитры  
- Адаптивный дизайн под разные экраны
